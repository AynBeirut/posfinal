## Troubleshooting

**Scanner not working?**
- Ensure scanner is in keyboard emulation mode
- Check USB connection
- Try manual entry first (F2)

**Wrong product added?**
- Check barcode matches product
- Update barcode in Admin panel

**No beep sound?**
- Browser may block audio on first use
- Click anywhere on page first
- Check system volume
